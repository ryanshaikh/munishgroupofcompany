# Munish Group | Operations Toolkit

Professional administrative automation portal for internal real estate operations.

## 🚀 Deployment
1. Upload all `.html` files to the root directory.
2. Enable **GitHub Pages** under Repository Settings.
3. Access via the primary `index.html` secure gateway.

## 🔐 Security Protocols
- **Authentication**: Required for form access.
- **Data Handling**: Forms are client-side processed; no data is stored on GitHub.
- **Print Logic**: Optimized for **ISO A4 Portrait** with zero-gap fluid typography.

## 📁 File Manifest
- **C / BC**: Static offline forms for handwritten entries.
- **EC / EBC**: Dynamic editable forms with receipt generation and interactive checkboxes.
